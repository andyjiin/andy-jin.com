imhungry

A sentient resturant finder

TECHNOLOGY: Amazon Alexa | AWS Lambda | IBM Watson | Firebase | Google Maps API | Yelp API
LANGUAGE: Python | Javascript | HTML/CSS

    •         imhungry is a project developed for Hack the Valley 2, a 3 day hackathon in Feburary 2018, with a team consisting of 3 other students; Arjun Bhushan (SE), Sara Fong (SE) and Marcus Yung (CE). The idea of the project was to incorporate Amazon Alexa's voice recognition algorithms with IBM Watson's Natural Language Processing in order to interpret mood to recommend food choices via Yelp API services.


            I worked primarily with incorporating IBM Watson's Natural Language Processing system with Amazon Alexa's speech recognition and then mapping the sentient scores produced to our backend which consisted of a server and database hosted on Firebase. The information relayed to the backend was then processed through Yelp API and displayed on the frontend via Google Maps API to show the user where the recommended resturants are located.
